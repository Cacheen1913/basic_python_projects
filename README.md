#PYTHON PROJECTS

A structured set of beginner-friendly to intermediate-level Python projects showcasing automation, web apps, AI integration, GUI tools, and backend logic.
Each project is designed to build strong programming foundations with hands-on implementation.

> Project List

1. Calculator

A simple command-line calculator supporting basic arithmetic operations like addition, subtraction, multiplication, and division.

2️. ATM PIN System

A console-based ATM system that simulates PIN verification, balance inquiry, withdrawal, and mini-banking operations.

3️. Unit Converter

Converts units such as length, weight, temperature, etc., using Python functions and conditional logic.

4️. SMS Automation

Automates sending SMS messages using available APIs. Useful for notifications and scheduled messaging tasks.

5️. Email Automation

A Python script to automate sending emails with attachments, HTML templates, and secure authentication.

6️. WhatsApp Automation

Uses Python automation libraries to send WhatsApp messages programmatically at specific times.

7️. Tic Tac Toe Game

A two-player Tic-Tac-Toe game playable in the console with winning logic and draw detection.

8️. Streamlit Dashboard

An interactive dashboard built using Streamlit, showcasing charts, inputs, and real-time UI updates.

9️. Text-to-Speech Tool

Converts input text into speech using Python’s TTS libraries, supporting various voice outputs.

10. Positive–Negative Checker

A simple script that checks whether a given number is positive, negative, or zero.

11. Movie Ticket Booking System

Simulates a ticket booking system where users can select movies, seats, and generate booking confirmation.

12. AI Chatbot (Gemini API)

An AI-powered chatbot built using Google’s Gemini model, capable of answering questions and generating responses.

13. Resume Builder using Flask

A Flask web application where users can input their details and generate a clean, downloadable resume.

14. Personal File Transfer System

A Python-based tool to securely transfer files between folders or devices in a structured way.

> Tech Stack Used
